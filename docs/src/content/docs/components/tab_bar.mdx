A `TabBar` displays a row of tabs for navigation between different views or content sections.

## Properties

### current_item
<SlintProperty propName="current_item" typeName="int" propertyVisibility="in-out">
The index of the currently selected tab.
</SlintProperty>

### items
<SlintProperty propName="items" typeName="[NavigationItem]">
An array of tab items to display in the bar.
</SlintProperty>
