A `TimePicker` allows users to select a time using a clock interface or text input.

## Properties

### hour
<SlintProperty propName="hour" typeName="int" propertyVisibility="in-out">
The selected hour (0-23).
</SlintProperty>

### minute
<SlintProperty propName="minute" typeName="int" propertyVisibility="in-out">
The selected minute (0-59).
</SlintProperty>

### read_only
<SlintProperty propName="read_only" typeName="bool">
Whether the time picker is read-only.
</SlintProperty>

## Callbacks

### accepted()
Invoked when the time selection is confirmed.

### cancelled()
Invoked when the time selection is cancelled.
