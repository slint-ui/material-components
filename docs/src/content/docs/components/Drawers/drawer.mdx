A `Drawer` is a side panel container that provides a structured layout for navigation or content. It includes an optional header with a title and a content area for custom elements.

## Properties

### title
<SlintProperty propName="title" typeName="string">
The title text displayed in the drawer header. If empty, no header is shown.
</SlintProperty>

## Behavior

- The drawer has a rounded corner design with material design styling
- Content is laid out vertically with proper spacing and padding
- The drawer automatically sizes to fit its content with a minimum width of 360px
- The header is only displayed when a title is provided
- Custom content can be added as children and will appear below the header
