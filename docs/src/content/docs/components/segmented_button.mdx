A `SegmentedButton` displays a group of related options as a single control, where only one option can be selected at a time.

## Properties

### current_item
<SlintProperty propName="current_item" typeName="int" propertyVisibility="in-out">
The index of the currently selected item.
</SlintProperty>

### model
<SlintProperty propName="model" typeName="[SegmentedItem]">
An array of segmented items, each containing an icon and text.
</SlintProperty>

## Functions

### select(index: int)
Selects the item at the specified index.
